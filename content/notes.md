# Using gdb to debug vis

~~~
$make debug
$gdb ./vis
(gdb) break prompt_enter
(gdb) run
...
(gdb) step // step (into) the next command
(gdb) next // go to the next line
(gdb) print s // prints a string
(gdb) print *arg // prints the content of a pointer to a structure
~~~
